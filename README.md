<h2>🛠️ Features</h2>

<ul>
  <li>🔍 Search by LeetCode username</li>
  <li>📊 Visual progress indicators for Easy, Medium, Hard problems</li>
  <li>📈 Stats cards for total solved, acceptance rate, ranking</li>
